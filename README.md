# ⚽ Estoril Praia — Bootstrap Redesign

> Same club. Better code.

A rebuild of my previous Estoril Praia site, this time using Bootstrap 5.
Same theme, completely different approach to layout and components.

## What changed from v1

| Feature | v1 (Pure CSS) | v2 (Bootstrap) |
|---|---|---|
| Layout | Manual Flexbox | Bootstrap Grid |
| Navbar | Built from scratch | `navbar` component |
| Responsiveness | Partial | Built-in |
| Forms | None | Bootstrap form controls |
| Icons | SVG via CDN | Bootstrap Icons / FontAwesome |

## What's inside

- Bootstrap 5 grid system and utility classes
- Responsive navbar with brand logo
- Image gallery with consistent `aspect-ratio` cropping
- Contact section with form and embedded Google Maps
- Fixed "back to top" button
- Google Fonts (`Orbitron`) combined with Bootstrap
- Custom CSS layer on top of Bootstrap for club branding
  (yellow `#FFEE00` + blue — very on brand)
- `min-height: 100vh` sticky footer pattern
